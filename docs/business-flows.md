# 校园 OA 核心业务流程清单

## 学生事务

| 流程 | 发起人 | 审批链 | 关键输出 |
| --- | --- | --- | --- |
| 请假 | 学生 | 班主任/辅导员 -> 学院 | 请假台账、消息通知 |
| 销假 | 学生 | 班主任/辅导员 | 销假确认记录 |
| 实习材料 | 学生 | 班主任 -> 学院 -> 教务备案 | 实习材料归档 |
| 异常学生材料 | 班主任 | 学院 | 跟踪记录、干预闭环 |
| 奖助学金 | 学生 | 班主任 -> 学院 -> 学工处 | 评定结果 |
| 证明开具 | 学生 | 班主任/学院 | 可打印证明单 |
| 学生预警 | 系统/班主任 | 学院 | 预警台账 |

## 教学教务

| 流程 | 发起人 | 审批链 | 关键输出 |
| --- | --- | --- | --- |
| 教师调课 | 任课教师 | 教研室 -> 学院 -> 教务处 | 调课通知 |
| 教材征订 | 任课教师 | 教研室 -> 学院 -> 教务处 | 征订汇总表 |
| 课程标准评审 | 任课教师 | 教研室 -> 学院备案 | 评审意见和结论 |
| 教案/大纲提交 | 任课教师 | 教研室 | 教学资料归档 |
| 成绩更正 | 任课教师 | 学院 -> 教务处 | 成绩更正记录 |
| 毕业设计 | 教师/学生 | 教研室 -> 学院 | 过程材料和结果 |
