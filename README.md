# 校园 OA 系统

本项目用于建设校园协同办公平台，服务对象覆盖学生、班主任、任课教师、教研室、学院、教务处、科技处、后勤部门和系统管理员等角色。系统目标是将校园内高频线下流程逐步信息化，实现统一入口、统一权限、统一流程、统一附件、统一消息和统一统计。

## 项目目标

- 建设校园 OA 平台基础能力，包括用户、角色、组织、菜单、权限、流程、附件、消息、日志
- 支撑学生事务、教学教务、科研项目、行政后勤四大业务域
- 优先落地高频业务流程，如请假销假、实习材料、异常学生、课程标准评审、课题申报、会议室申请、维修工单等
- 为后续移动端、统一身份认证、消息推送、对象存储和复杂工作流引擎预留扩展空间

## 当前技术方案

- 前端：`Vue 3 + Vite + TypeScript + Element Plus`
- 后端：`Spring Boot`
- 数据库：`MySQL`
- 文档：`Markdown`

## 技术栈说明

为保证文档、代码和后续开发方向一致，当前项目统一按以下技术基线执行：

- 前端主栈：`Vue 3 + Vite + TypeScript + Element Plus`
- 前端配套：`Pinia + Vue Router`
- 后端主栈：`Spring Boot`
- 后端配套：`Spring Security + MyBatis-Plus`
- 数据库：`MySQL`
- 文档：`Markdown`

## 本地启动说明

### 后端

1. 安装 `JDK 17` 和 `Maven 3.9+`
2. 创建数据库 `campus_oa`
3. 执行 `database/mysql/001_init_schema.sql`
4. 修改 `backend/src/main/resources/application.yml` 中的数据库配置
5. 在 `backend` 目录执行 `mvn spring-boot:run`

### 前端

1. 安装 `Node.js 20+`
2. 在 `frontend` 目录执行 `npm install`
3. 执行 `npm run dev`
